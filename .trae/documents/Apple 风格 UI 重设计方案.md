# Apple 风格视觉重设计计划

根据您的要求，我将把现有的 UI 全面升级为 Apple 官网风格。核心设计理念是：**极简、沉浸式、内容优先**。

## 1. 全局样式与基础架构 (Global Styles)
- **背景色**: 使用 Apple 经典的浅灰背景 (`bg-[#f5f5f7]`) 代替当前的渐变色。
- **字体**: 强化系统字体栈（San Francisco 风格），优化字重和抗锯齿效果 (`antialiased`)。
- **色彩体系**: 
  - 主文本: `#1d1d1f` (接近纯黑的深灰)
  - 次级文本: `#86868b` (中性灰)
  - 强调色: `#0071e3` (Apple Blue)
- **布局容器**: 增加页面留白，使用更宽的圆角 (`rounded-2xl` 到 `rounded-3xl`)。

## 2. 顶部导航栏 (Header)
- **Glassmorphism**: 采用半透明毛玻璃效果 (`backdrop-blur-md bg-white/70`)，固定在顶部。
- **布局**: 缩小高度（48px-64px），内容居中，Logo 和标题更精致、低调。

## 3. 核心区域 (Hero Section)
- **排版**: 标题采用超大字号 (`text-5xl` 或 `text-6xl`)，紧凑字间距 (`tracking-tight`)，居中对齐。
- **副标题**: 使用较大的字号和灰色调，增加行高以提升可读性。
- **视觉重心**: 移除复杂的背景装饰，让文字本身成为视觉焦点。

## 4. 智能查询工具 (Search Component)
- **卡片化**: 将搜索区域封装在一个白色大卡片中 (`bg-white rounded-3xl shadow-sm`)。
- **输入框**: 使用浅灰色背景 (`bg-[#f5f5f7]`) 的圆角输入框，无边框设计。
- **按钮**: 改为胶囊形按钮 (`rounded-full`)，使用 Apple Blue，添加平滑的悬停缩放效果。

## 5. 结果展示 (Results)
- **Bento Grid 布局**: 将查询结果从简单的列表改为类似 Apple 产品特性的 "Bento" 风格网格。
- **数据可视化**: 每个核心指标（如蓝思值、CEFR）独立占据一个小卡片，突出数字本身。

## 6. 对标表 (Comparison Table)
- **现代化表格**: 移除垂直分割线，仅保留柔和的水平分割线。
- **交互**: 表格行添加悬停高亮，整体容器采用大圆角和白色背景。

## 7. 底部 (Footer)
- **极简设计**: 浅灰背景，小字号文本，去除多余的装饰，仅保留版权和必要链接。
